# Visual Studio 好用設定與技巧

- Shift+Alt+.
  - 往下選取相同的文字
- Shift+Alt+;
  - 全選相同的文字
- Ctrl+Shift+insert
  - 循環剪貼簿(GUI)
- Shift+Enter
  - 文字插入點跳到下一行
- Ctrl + F12
  - 移動至實作

# Visual Studio 文字設定

* 選項 &rarr; 環境  &rarr; 字型和色彩  &rarr; 字型下拉框  &rarr; 選擇 Cascadia Code 大小 12 
  - 這樣設定對眼睛比較舒服XD
