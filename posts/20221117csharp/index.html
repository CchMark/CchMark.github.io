е·ҘдҪңдёҠеёёиҒҪеҲ°DapperпјҢдҪҶдёҚзҹҘйҒ“йҖҷеҖӢеҘ—д»¶еҜҰйҡӣеҰӮдҪ•дҪҝз”ЁпјҢжүҖд»ҘеҜ«дёҖзҜҮеӯёзҝ’зӯҶиЁҳXD

Dapperд»Ӣзҙ№

DapperжҳҜMicrosoft .NETе№іеҸ°зҡ„зү©д»¶й—ңиҒҜе°ҚжҮүORM(object-relational mapping)еҘ—д»¶пјҡе®ғжҸҗдҫӣдәҶдёҖеҖӢе°Үе°ҚиұЎиіҮж–ҷжҳ е°„еҲ°иіҮж–ҷеә«зҡ„жЎҶжһ¶гҖӮDapper жҳҜе…ҚиІ»зҡ„й–Ӣжәҗи»ҹд»¶пјҢеңЁLicenseйғЁеҲҶзӮәApache License 2.0жҲ–MIT License еҸҜд»Ҙе®үеҝғдҪҝз”ЁXD

  • Dapper жҳҜйҒ©з”Ёж–ј .NET зҡ„й–Ӣжәҗе°ҚиұЎй—ңдҝӮжҳ е°„ (ORM) еә«гҖӮ
  • йҖҡйҒҺе°Үе°ҚиұЎжҳ е°„еҲ°иіҮж–ҷеә«дёӯзҡ„иЎЁпјҢеҸҜд»Ҙиј•й¬Ҷиҷ•зҗҶжҮүз”ЁзЁӢеәҸдёӯзҡ„иіҮж–ҷгҖӮ
  • Dapper еҝ«йҖҹгҖҒеҸҜйқ дё”з¶“йҒҺе……еҲҶжё¬и©ҰпјҢеӨҡе№ҙдҫҶе·Іиў«дё–з•ҢдёҠдёҖдәӣжңҖеӨ§зҡ„е…¬еҸёз”Ёж–јз”ҹз”ўгҖӮ
  • е®ғйқһеёёжҳ“ж–јдҪҝз”ЁпјҢдёҰе…·жңүиЁұеӨҡеҠҹиғҪпјҢдҪҝе…¶жҲҗзӮәиіҮж–ҷеә«иЁӘе•Ҹзҡ„еј·еӨ§е·Ҙе…·гҖӮ

дё»иҰҒеҠҹиғҪеҢ…еҗ«:

Speedy and high performance(еҝ«йҖҹдё”й«ҳж•ҲиғҪ)

Fewer lines of code(жёӣе°‘еҜ«codeиЎҢж•ё)

Object mapper(е°ҚжҮүзү©д»¶)

Choice of static/dynamic object binding(з¶Ғе®ҡйқңж…ӢжҲ–еӢ•ж…Ӣзү©д»¶)

Easy handling of SQL query(з°Ўе–®SQLиҷ•зҗҶ)

Multiple query support(ж”ҜжҸҙеӨҡзӯҶиҷ•зҗҶ)

Support and easy handling of stored procedures(ж”ҜжҸҙдё”иј•й¬Ҷзҡ„иҷ•зҗҶй җеӯҳзЁӢеәҸ)

Operating directly on IDbConnection interface(зӣҙжҺҘдҪҝз”ЁIDbConnectionд»Ӣйқў)

  • Dapper IDbConnectionжҸҗдҫӣжңүз”Ёзҡ„д»Ӣйқўж“ҙеұ•ж–№жі•дҫҶжҹҘи©ўиіҮж–ҷеә«гҖӮ
  • е®ғдҪҝз”ЁеӢ•ж…Ӣж–№жі•з”ҹжҲҗдҪҝе…¶иғҪеӨ зӣҙжҺҘеҫһжҹҘи©ўзөҗжһңдёӯж“ҙе…… POCOsгҖӮжӯӨеӨ–пјҢе®ғйӮ„е…ҒиЁұжӮЁе°ҮиіҮж–ҷеә«еҲ—зӣҙжҺҘжҳ е°„еҲ° POCO дёҠзҡ„еұ¬жҖ§гҖӮ
    • дҪҝз”Ё Dapper жҷӮпјҢеҸӘйңҖиҰҒдёҖеҖӢйҖЈжҺҘеӯ—з¬ҰдёІе’ҢдёҖеҖӢ POCOпјҢз„¶еҫҢе°ұжҳҜдёүжӯҘжөҒзЁӢгҖӮ
      • еүөе»әдёҖеҖӢIDbConnectionе°ҚиұЎгҖӮ
      • з·ЁеҜ«жҹҘи©ўд»Ҙеҹ·иЎҢ CRUD ж“ҚдҪңгҖӮ
      • ExecuteеңЁд»»дҪ•жҲ–Queryж–№жі•дёӯе°ҮжҹҘи©ўдҪңзӮәеҸғж•ёеӮійҒһгҖӮ

Bulk data insert functionality(еӨ§йҮҸиіҮж–ҷж–°еўһ)

Dapper ж–№жі•жҮүз”Ё

Dapper ж“ҙеұ•ж–№жі•еҸҜз”Ёж–јеңЁиіҮж–ҷеә«дёӯеҹ·иЎҢеҗ„зЁ®ж“ҚдҪңпјҢдҫӢеҰӮзҚІеҸ–ж•ёж“ҡгҖҒжҸ’е…ҘиЁҳйҢ„гҖҒжӣҙж–°иЁҳйҢ„е’ҢеҲӘйҷӨиЁҳйҢ„гҖӮдёҰе°ҮдҪҝз”ЁеӨҡзЁ®ж–№жі•е’ҢдёҖдәӣжңҖеёёз”Ёзҡ„ dapper ж“ҙе……ж–№жі•дҫҶж“ҙе……IDbConnectionд»Ӣйқў

  • Execute: е®ғеҸҜд»Ҙеҹ·иЎҢдёҖж¬ЎжҲ–еӨҡж¬Ўе‘Ҫд»ӨгҖӮ

  • ExecuteReaderпјҡе®ғеҸҜд»Ҙеҹ·иЎҢдёҖеҖӢе‘Ҫд»ӨдёҰиҝ”еӣһдёҖеҖӢreaderгҖӮ

  • Executescalarпјҡе®ғеҸҜд»Ҙеҹ·иЎҢдёҖеҖӢе‘Ҫд»ӨдёҰиҝ”еӣһдёҖеҖӢжЁҷйҮҸеҖјгҖӮ

  • Query: з”Ёж–јеҫһж•ёж“ҡеә«дёӯзҚІеҸ–ж•ёж“ҡгҖӮ

  • QueryFirst: QueryFirstпјҡе®ғеҸҜд»Ҙеҹ·иЎҢжҹҘи©ўдёҰжҳ е°„з¬¬дёҖеҖӢзөҗжһңгҖӮ

  • QueryFirstOrDefault: QueryFirstOrDefaultпјҡе®ғеҸҜд»Ҙеҹ·иЎҢжҹҘи©ўдёҰжҳ е°„з¬¬дёҖеҖӢзөҗжһңпјҢжҲ–иҖ…еҰӮжһңеәҸеҲ—дёҚеҢ…еҗ«е…ғзҙ еүҮжҳ е°„й»ҳиӘҚеҖјгҖӮ

  • QuerySingle: е®ғеҸҜд»Ҙеҹ·иЎҢжҹҘи©ўдёҰжҳ е°„з¬¬дёҖеҖӢзөҗжһңпјҢеҰӮжһңеәҸеҲ—дёӯдёҚеӯҳеңЁдёҖеҖӢе…ғзҙ еүҮжӢӢеҮәз•°еёёгҖӮ

  • QuerySingleOrDefault:QuerySingleOrDefaultпјҡе®ғеҸҜд»Ҙеҹ·иЎҢжҹҘи©ўдёҰжҳ е°„з¬¬дёҖеҖӢзөҗжһңпјҢеҰӮжһңеәҸеҲ—зӮәз©әеүҮзӮәй»ҳиӘҚеҖјпјӣеҰӮжһңеәҸеҲ—дёӯжңүеӨҡеҖӢе…ғзҙ пјҢжӯӨж–№жі•е°ҮжӢӢеҮәз•°еёёгҖӮ

  • QueryMultiple: е®ғеҸҜд»ҘеңЁеҗҢдёҖеҖӢе‘Ҫд»Өе’Ңжҳ е°„зөҗжһңдёӯеҹ·иЎҢеӨҡеҖӢжҹҘи©ўгҖӮ
